# NowPaymemts.io
# Dating Platform

A modern full-stack dating platform built with React, Node.js, Express, Sequelize, MySQL, Tailwind CSS, shadcn/ui, Redux Toolkit, Socket.IO, and NOWPayments.

# AI STARTUP CHECKLIST (MANDATORY)

Before writing or modifying any code, complete these steps in order:

1. Read `COPILOT_INSTRUCTIONS.md`.
2. Read `docs/PROJECT_SCOPE.md`.
3. Read `docs/PROJECT_PHASES.md`.
4. Read `docs/CHANGELOG.md`.
5. Read `docs/DATABASE_SCHEMA.md`.
6. Read `docs/API_DOCUMENTATION.md`.
7. Inspect the existing project structure and relevant files.
8. Search the project before creating any new file, component, utility, middleware, controller, route, model, hook, or Redux slice.
9. Confirm the requested task aligns with the current development phase unless the user explicitly requests work from another phase.
10. Implement only the requested feature or fix, making the smallest necessary changes.
11. Update `docs/CHANGELOG.md` after completing a feature or bug fix.
12. If a new endpoint, model, or database relationship is introduced, update the relevant documentation (`API_DOCUMENTATION.md` or `DATABASE_SCHEMA.md`).
